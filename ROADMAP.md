# 開発ロードマップ

## 📊 プロジェクト概要

このプロジェクトは、歴史的通貨価値を現在価値に変換するWebアプリケーションです。

**技術スタック**
- React + TypeScript + Vite
- Tailwind CSS
- CoinGecko API
- 静的CPIデータ
